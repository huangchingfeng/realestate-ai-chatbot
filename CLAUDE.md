# 房仲 AI 自動回覆模擬器 Demo

> 一句話：模擬房仲社群私訊自動回覆，展示 AI 如何即時回應客戶詢問

## 技術棧
- 純前端：HTML + CSS + Vanilla JS（單一檔案）
- AI 引擎：Google Gemini 2.0 Flash
- 無需後端，API Key 預填

## 功能
1. 物件資料庫管理（新增/編輯/標記已售出）
2. 經紀人資訊設定（姓名/公司/電話）
3. LINE / FB Messenger / IG DM 三平台模擬
4. AI 智能回覆（根據物件資料 + 對話歷史）
5. 快速問題按鈕（模擬常見客戶私訊）
6. 經紀人推播通知模擬
7. 已售出物件自動推薦類似物件

## 開發指令
```bash
open index.html
```
